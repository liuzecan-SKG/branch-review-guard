## 4. 高风险专题（由启用的规则包决定，如事务/并发/对外契约/数据迁移）

> 具体专题项由 `rules/` 中已启用的规则包注入（启用某栈包会补充该栈的机制级专题，如特定事务/锁注解、装配登记、上报协议等）；未启用栈包时本章只列通用高风险类别。

- 事务与原子性：多步写是否在同一事务、失败是否可回滚、跨存储是否一致
- 并发与锁：共享可变状态、check-then-act、锁 key/作用域、线程上下文不跨线程
- 对外契约：HTTP API / RPC / 消息队列 / 对接协议 的兼容性
- 数据迁移与配置：迁移脚本可回滚性、新依赖/新中间件登记、框架组件注册
